**ESP32**
 - Procesorius: 2 (dual core) Xtensa 32-bit LX6
 - Atmintis: RAM 512 KB; Flash memory; 
 - Wi-fi: 2.4 GHz up to 150 Mbits/s
 - Bluetooth 
 - BLE (Bluetooth Low Energy)
 - ESP-NOW (up to 250 bytes messages)
